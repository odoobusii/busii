Installation Instructions:

Once the module is installed, external customers will be able to view additional task details from the my Account portal via the new Project and Task tabs. These include start date, end date, dependencies, and order information — all visible through enhanced project and task cards, and detailed list views.

No additional setup is needed after installation. You can start sharing detailed project insights immediately from the portal.

Features:

Enhanced Project and Task Portal Views

Share better project updates with external users

Display task start and end dates directly in the portal for better visibility.

Show task dependencies via a "Blocked By" section on task detail pages.

Allow customers to see project and task cards with important information, improved grouping, and planned timelines.

New Project and Task Views for Customers

Empower external users with more project context

External customers now see improved task list views with sortable columns, planned dates, and more. Each task detail view also contains a breakdown of any blocking tasks.

Task Report via Project Tab

Easy visualization of your projects

In each task’s report document opened from the Project tab, information about the task's project is displayed.

Task Report with Dependencies Table via Task Tab

Easy visualization of what’s blocking your tasks

In each task’s report document, a clean and simple table lists all tasks it depends on, with links, dates, and descriptions. Help your clients keep track of delays and blockers.

⚠️ Important Disclaimer

If you encounter an error stating that “planned_date_begin” is not found on the Project Task model, or if the Planned Start Date field is missing on a task, then Odoo Studio (web_studio) must be installed. After installation, simply add the existing field planned_date_begin (Planned Start Date) to the Project Task form view for this module to function properly.
