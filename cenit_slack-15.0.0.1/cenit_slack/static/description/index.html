Slack authentication

After the module is install you should authorize Cenit's app to send messages to Slack in your behalf. Go to your Cenit's account and select Authorizations menu.

In the Slack authorization record, select Authorize action and authorize Cenit's app.


That's it! Every time a notification, email message or comment is created, it will be sent automatically to Slack.

Messages sent are related to Partner and Sales Orders. You can modify this behaviour and the fields sent to Slack. Check the mapping under Mappings tab in Cenit module and create your own mappings.
