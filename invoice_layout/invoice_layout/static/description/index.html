Installation Instructions:

After installing the module, navigate to Accounting → Customers Menu → Invoices and open any customer invoice. The modified layout will automatically reflect in the printed PDF version using the standard striped layout.

You don't need to activate or configure anything further—changes are applied directly to the invoice report template.

Features:

Enhanced Striped Invoice Layout

Cleaner Design and Better Payment Instructions

Replaces Tax ID with VAT no.

Adjusts layout spacing for improved document readability.

Custom message under the payment section with clearly visible payment reference.
