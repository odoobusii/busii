Installation Instructions

Once installed, this module automatically updates the PDF export layout of the Partner Ledger report in the Accounting module. No additional configuration is required.

Navigate to Accounting → Customers → Customers. Then choose a customer and select Customer Statement to open the Customer Statement form view. Click the PDF button to generate the updated PDF report.

Key Features

Company Logo: The company logo is now displayed in the top-left corner of the PDF.

Label Update: The filter label "Partners:" has been updated to "Customers:".

Customer Address: The address of the first selected customer is displayed, along with their Tax ID if available.

Visual Cleanup: The grey background in the header section has been removed for a cleaner, more professional look.
