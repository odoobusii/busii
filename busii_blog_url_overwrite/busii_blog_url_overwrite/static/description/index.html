Installation Instructions:

Once the module is installed on your server or Odoo.sh, your blog and post URLs will automatically update to a cleaner format without numeric IDs.

The updated URLs will follow this structure: /blog/blog-title/post-title, where both names are converted to lowercase and spaces replaced by hyphens.

Features:

Clean Blog URLs

Remove Unsightly IDs From Blog URLs

Automatically formats blog and post URLs to remove numeric IDs.

Supports live blog editing and applies custom URL logic dynamically.

Enforces lowercase and hyphenated formatting to prevent URL errors.

Blog Post Creation Integration

Real-Time URL Formatting in Blog Creation

When creating or editing blog posts, the system ensures your blog and post names are automatically converted into clean, SEO-friendly URLs.

Validation and Error Messaging

Avoid Improper Formatting

If blog or post titles include uppercase characters or invalid formatting, users will see a clear error message prompting them to use lowercase and hyphens only.
